16陈贤妹，广东省清远市阳山县七拱镇岩口村人。2011年10月13日，一名两岁零四个月的女童王悦在广佛国际机电五金城一街道先后被一辆面包车和汽车两次碾压，18名路人视而不见、见死不救，陈贤妹救起女童并找到孩子家长。

一个极其普通的老人，在救起先后遭受两辆汽车碾压的女童小悦悦，因与18个路人未施援手的走过形成强烈对比，一时间备受媒体和社会关注。继“最美妈妈”、“最美警察”、“最美乡村女教师”之后，陈贤妹这个满脸皱纹、年近六旬的乡村阿婆被冠以“最美婆婆”的美誉，这并不是网友的一时兴起，而是一种感动与尊重的延续。

女童被撞无人问津 拾荒阿姨施援手称对得起良心

陈贤妹：那天下午，我正在五金城里捡垃圾，远远看到一个小孩儿躺在路上，就赶忙走过去。我走到女仔旁边，听到她“啊啊”的叫声。我把孩子抱起来，看到她一只眼睁着，一只眼闭着，流着眼泪，嘴里和鼻子都在流血，后脑勺也在流血。我想把她抱起来，但孩子软绵绵的，马上就瘫了下去。我不敢用力，便把孩子拖到路边，喊路过的人帮忙，没有人理我。[详细]

18名过路人回应小悦悦

可怜的小悦悦，你走了。你这一走震动了那天在你身旁路过的18名行路人，更震动了大半个中国。当我们18人，当全中国的人民在电视上看到你被碾压的视频时，我们都掉泪了。我们也假设了：假如我们的孩子被车辆碾压没人施救时怎么办？今天是小悦悦，明天该是谁家的孩子？我们18名行路人和全中国的人民都想通了：危难关头，救人要紧。在生死存亡的关键时刻，我们要向陈贤妹阿姨那样，宁愿冒着被诬陷的风险也要救回幼小的生命。[详细]

一个人战胜十八个人的道德力量

18名路人见死不救,58岁的拾荒人陈贤妹是一位朴实的老人,身体瘦小,看起来弱不禁风,然而就是这样一位生活在社会最底层的拾荒老人,经常在垃圾和废物中穿行的人，却有一颗金子一样善良的心灵。从表面上看，陈贤妹是弱小的，是平凡的，这在人们普遍感觉世风日下、道德滑坡的当下,她的行为无疑是一声呐喊,她身上缭绕着的正气，战胜了18个人的冷漠和无奈,她用实际行动告诉人们,人间自有真情在。[详细]

网友作诗：扪心自问——小悦悦事件有感

路见不平千夫指， 油瓶倒地无人扶。 要求别人是君子， 唯独自己做小人。[详细]

叩问心灵 树立正确道德观

去了天堂的小悦悦依然叩问着我们每个人的良知

我们无需指责这个社会怎样的丑陋，我们无需指责别人怎样冷漠麻木，因为这里面包括你也包括我。我们只能自省，我们只能自律，我们只能从自己做起，不做丑恶的制造者。[详细]

省察我们的灵魂 让善意充实内心

温暖太稀少，爱是难的。在一个传统价值被颠覆，而新的价值却又付诸阙如的时代，省察灵魂，检讨现实，只能从你我自己的内心开始。全无相互热爱的生活，一片荒漠，不值一过。[详细]

我们该怎样重建中国式道德观？

我国文化发展同经济社会发展和人民日益增长的精神文化需求还不完全适应，突出矛盾和问题主要是：一些地方和单位对文化建设重要性、必要性、紧迫性认识不够，文化在推动全民族文明素质提高中的作用亟待加强。[详细]

用良知的尖刀刮骨疗伤

“小悦悦事件”中，一位在常人眼中属于“底层人”的拾荒阿姨，让泱泱大国的国人，保留了起码的颜面。向冷漠和“事不关己，高高挂起”的心态这个“怪胎”动手吧！让我们“拒绝冷漠，唤醒真爱”，用真爱颠覆冷漠。[详细]

不做恶的抱怨者要做善的行动者

见死不救不能止于道德谴责，而是应该积极反思；但是，所谓反思绝不简单等同为抱怨恶。如果我们的反思只是止于抱怨恶，那么善永远也不会成为有力的行动。[详细]

拒绝冷漠 传递温暖

小悦悦事件似乎过去了，留在人们心头的痛也似乎在慢慢愈合，每天都会有各种各样的事件发生，每天都会有各种热点消息充斥各个媒体上，比起那些离我们很远的东西，拒绝冷漠，传递温暖这种事情还是比较实际吧。[详细]

好人好事 传递爱心温暖

北京79路公交车司机5年背患病老人上下车

“请其他乘客先等一等，让老人家先上车。”某日上午9点多，海淀田村半壁店车站，79路公交车司机李英庆稳稳地停好车后，一边示意其他乘客让老人先行，一边下车熟练地背起路边候车的焦大爷。这位老人和李师傅没有血缘关系，但老人却说，“他就像我的亲儿子一样亲”。在李师傅车里的乘客留言本上，有人写道：“一个人做一件好事不难，难的是一辈子做好事。79路司机平凡的坚持，让人温暖，让人感动。” [详细]

女孩遇车祸被卷入车底 20多人抬起车将其救出

近期发生在广东的“小悦悦”事件击中了社会之痛，激发了人们对道德和爱心的思考。然而19日发生在南昌的一起车祸，让我们又看到了这个社会并不缺少关爱：面对一名被压在车底的素不相识的女孩，二十多人硬生生抬起一吨重的轿车，救出了女孩，他们说，“我们不能见死不救！”面对需要帮助的人，部分人选择冷漠观望，可发生在小巩身上的事情，又再一次证明社会上还是存在爱心和真情的。 [详细]

救病倒在地老太太，3位学妹好样的！

82岁的杨老太太外出购物，因病跌坐在板凳上，并不住地呕吐，路过的无为一中学生蒋颖琦、李云雅、季丹丹发现后，来不及多想，简单问明情况后，开始用她们学过的简单施救方法对老人施救，并将她扶到了学校警卫室，在附近药房买来药给老太太服下。随后拨通了120，三位同学还随120将老太太送到医院抢救，直到三个小时后，老太太病情稳定，她们才悄悄地离开医院，赶往学校。[详细]

用行动证明奉献 用良知温暧社会

丰都县太平坝乡女童刘天艳、杨夕梅一道去蓄水池玩耍，刘天艳不慎脚一滑掉进了水池里，岸上的杨夕梅大声哭喊着“救命”。此时，就读于重庆大学2007级化学应用系的刘波涛刚回到家20分钟。正在吃饭的他听到呼救声后，立即丢下饭碗疾步冲到水池边。刘波涛虽不会游泳，但他没有丝毫犹豫便跳进水池。[详细]

记住那些平凡的善良和感动

陈阿婆的善举让人们心生感动，人间自有真情在。在我们的生活中，也不乏这样平凡的善良与感动。徒手接住坠楼女童的“最美妈妈”吴菊萍；为救落水孩童献出宝贵生命的“最美爷爷”占祖亿、“最美奶奶”柴小女；嘴对嘴为溺水老人做人工呼吸的90后“最美女孩”余书华……他们是社会中平凡却善良的大多数。[详细]

坐而论道，不如起而前行。时代呼唤更多“陈贤妹”，期待涌现更多“最美”。让我们行动起来，从自己做起，以捍卫道德和人性的名义，拒绝冷漠，见义勇为，乐于助人。
